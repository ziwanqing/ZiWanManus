# 🧠 ZiWan AI Agent 项目介绍

## 项目概述

本项目是一个基于 **Java 21 + Spring Boot 3** 框架的 **智能体开发与AI大模型应用平台实战项目**，围绕 **Spring AI / Spring Al 开发框架** 构建。项目选题新颖、业务真实，旨在让开发者掌握从 AI 大模型接入到智能体自主开发的完整流程。

项目核心目标：

- 打通 **AI大模型接入、开发、部署与应用** 全流程
- 构建 **RAG知识库** 支撑的智能问答系统
- 实现 **Tool Calling 工具调用** 与 **MCP模型上下文协议**
- 掌握 **Spring Al 核心特性**：对话记忆、结构化输出、自定义 Advisor
- 提供高性能、异步和多模态的 AI 服务

------

## 核心功能

### 1. AI大模型接入与应用

- 支持主流 AI 平台及大模型接入，包括云端和本地部署
- 支持四种大模型接入方式（API、SDK、本地部署、嵌入式调用）
- 提供 **Prompt 工程和优化技巧**，提升模型响应质量

### 2. Spring Al 开发框架

- 自定义 **Advisor** 实现对话策略和行为控制
- **对话记忆管理**，保持多轮上下文
- **结构化输出**：统一模型输出格式，便于系统处理

### 3. RAG 知识库与向量数据库

- 基于 **PgVector 向量数据库** + 云数据库服务，实现知识向量化检索
- 支持知识库构建、检索、查询优化和调优技巧
- 与大模型结合，实现高精度问答和知识增强生成

### 4. Tool Calling 与 MCP 协议

- 实现 **多工具调用（SearchAPI、PexelsAPI 等）**
- 支持 **MCP（模型上下文协议）**，在智能体间传递上下文
- 构建 **ReActAgent 智能体**，实现自主任务执行

### 5. 高级技术与新概念

- 多模态处理（文本、图像、PDF 等）
- SSE 异步推送，实现实时交互
- 智能体工作流和 A2A 协议设计
- 大模型评估与性能优化

### 6. 技术栈

- **语言与框架**：Java 21, Spring Boot 3, Spring Al,
- **数据库**：PostgreSQL + PgVector, Redis, MongoDB, MySQL
- **工具库**：Kryo（高性能序列化）、Jsoup（网页抓取）、iText（PDF生成）、Knife4j（接口文档）
- **第三方大模型平台**：百炼AI开发平台

------

## 项目亮点

- **完整AI开发流程**：从大模型接入、Prompt优化、知识检索、到智能体执行
- **自主智能体构建**：支持多Agent协作与上下文管理
- **高可扩展性**：工具调用、异步SSE推送
- **实际业务落地**：可在智能问答、文档处理、图像生成等场景中应用
